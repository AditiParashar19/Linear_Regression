# Linear Regression

This repository contains multiple implementations and applications of **Linear Regression**, built from scratch and using real-world datasets. It focuses on understanding both the **mathematics** and **practical implementation** of machine learning models.

## Projects File Included

### Beijing PM2.5 Linear Regression
- Predicts air pollution levels (PM2.5)
- Works with real-world environmental data
- Includes data preprocessing and feature selection

###  Linear Regression From Scratch
- Implementation without using ML libraries
- Covers:
  - Cost Function (MSE)
  - Gradient Descent
  - Parameter updates

###  Linear Regression (Maths + GenAI)
- Focus on mathematical understanding
- Step-by-step derivation of formulas
- Visualization of learning process

###  Obesity Dataset Regression
- Predicts obesity-related outcomes
- Includes feature engineering and model evaluation

## Concepts Covered

- Simple Linear Regression  
- Multiple Linear Regression  
- Gradient Descent Algorithm  
- Cost Function (Mean Squared Error)  
- Feature Scaling  
- Model Evaluation
